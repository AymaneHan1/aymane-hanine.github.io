Mass spectrometry is a physics technique used to identify molecules by analyzing characteristic spectra of chemical structures. However, experimental spectra are often noisy, making it challenging to identify chemical components. Additionally, machine learning applications such as spectrum classification and component identification require large, realistic databases of spectra to achieve effective results. This project has two main objectives: proposing methods for creating realistic databases and denoising spectra using deep learning techniques. We obtained spectra through three different techniques: the European MassBank project database, the Pyisopach library, and random generation. After creating three realistic databases, we developed a neural network architecture called an autoencoder, which we trained and optimized to denoise signals. Finally, we built a website to make this work accessible to everyone.
