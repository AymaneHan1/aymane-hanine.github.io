  • Spam Email Classification:: In this project, I developed a deep learning model to predict whether an email is spam or non-spam using "bags of words" features. I explored various algorithms such as deep neural networks (DNN), convolutional neural networks (CNN), logistic regression (LR), support vector classifier (SVC), polynomial classifier (PolyC), random forest classifier (RFC), and gradient boosting classifier (GBC). I evaluated their performance using metrics like recall and F1 score, ultimately identifying the best model for spam detection. This project demonstrates my expertise in natural language processing and machine learning for classification tasks.
  • Ozone Concentration Prediction:: The objective of this project was to predict the concentration of ozone based on meteorological conditions and the previous day's ozone concentration. I implemented several prediction algorithms including manually defined neural networks (DNN), convolutional neural networks (CNN), linear regression (LR), support vector regression (SVR), polynomial regression (PR), random forest (RF), and gradient boosting regression (GBR). I preprocessed the data by handling missing values, outliers, and analyzing feature correlations. By comparing the performance of these models using metrics like mean absolute error (MAE), I determined the most effective model for ozone concentration prediction. This project showcases my skills in regression analysis and predictive modeling.
